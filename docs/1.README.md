# 🏢 Sistema de RRHH - Control de Permisos y Vacaciones

Sistema integral para administrar días de vacaciones, ausencias y permisos de los empleados de Viajar LTDA. Construido full-stack con React, Vite, Node.js, Express y SQLite.

## 🚀 Características Principales

1. **Cálculo de Días Inteligente con Saltos de Festivos**: 
   - El sistema calcula automáticamente los días ganados en base a la **fecha de ingreso**. 
   - Excluye domingos y **festivos nacionales y regionales (Colombia/Barranquilla)** automáticamente al solicitar permisos mediante el algoritmo de Ley Emiliani implementado.
   - Soporte para 7 tipos de ausencias: *Vacaciones, Día Libre, Sábado Compensatorio, Calamidad, Incapacidad, Cita Médica, y Licencia No Remunerada*.

2. **Tipos de Permisos Extendidos y Soporte Documental**:
   - Clasificación híbrida: Tipos que descuentan saldo (Vacaciones, Día Libre, Sábados) y tipos que se acumulan progresivamente (Incapacidades, Calamidad, etc.).
   - Permite adjuntar archivos de **Soporte Documetal (PDF, Imágenes, etc.)** directamente desde la vista del Colaborador o Director, consolidándolo en las tablas de RRHH.

3. **Jerarquía de Aprobación Avanzada**:
   - **Flujo Estándar**: Colaborador → Director Directo.
   - **Suplencia Dinámica**: Si un Director está de vacaciones, el **Gerente** puede visualizar y aprobar las solicitudes de su equipo.
   - **Autogestión Gerencial**: El Gerente solicita sus permisos y RRHH debe aprobarlos, y viceversa.
   - Nueva nomenclatura de roles implementada: *Admin, RRHH, Gerente, Director, Colaborador*.

4. **Panel Administrativo de RRHH de Última Generación**:
   - **Edición Integral**: Permite modificar perfiles completos (DNI/Documento, Email, Cargo, Director, Fecha Ingreso) ajustando directamente toda la información en los grid-layouts de balance.
   - **Filtros Híbridos Inteligentes**: Búsqueda por texto libre combinada con *documento de identidad* y estados para Nombres, Correos y Cargos sin que el usuario actual desaparezca.
   - **Exportación a Excel/CSV**: Generación de reportes limpios, incluyendo todos los tipos de permisos solicitados listos para descarga rápida.

5. **Experiencia de Usuario (UI/UX) Pro**:
   - Paridad visual lograda en paneles cruzados (El panel de solicitudes del Director y Gerente es idéntico a la experiencia óptima del Colaborador).
   - Descargas de Soportes usando `<a download>` para proteger la experiencia transaccional.

## 🛠️ Guía de Inicio Rápido

### 1. Requisitos
- Node.js instalado.
- Terminal para Backend y Terminal para Frontend.

### 2. Instalación
```bash
# Backend
cd backend && npm install && npm run dev
# Frontend (en otra terminal)
cd frontend && npm install && npm run dev
```
